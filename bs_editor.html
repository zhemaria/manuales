01 Introducción al Editor de Brightspace +

El Editor de Brightspace permite crear e insertar contenido dentro de Brightspace. Se integra con todas las herramientas del entorno de aprendizaje que tienen capacidades de creación de contenido HTML, como los temas de discusión, las instrucciones personalizadas para carpetas de tareas, los temas de contenido y más.

Bueno saberlo

Al crear o editar archivos de contenido con el editor existe un límite de 2.5 millones de caracteres por archivo.

02 Opciones de visualización de contenido +

El Editor de Brightspace ofrece tres maneras de ver y trabajar con tu contenido:

Opciones de visualización del editor y su uso recomendado.
Opción de vista Uso
Vista de Diseño Se carga automáticamente al acceder al editor. Úsala para crear y dar formato al contenido con rapidez, sin necesidad de conocer HTML. Permite escribir, aplicar formato, insertar imágenes y tablas, y crear enlaces con los controles disponibles.
Editor de Fuente (código HTML) Se accede con el ícono del Editor de fuente de HTML. Muestra el código HTML que estructura y formatea el contenido. Es ideal si tienes experiencia con HTML o deseas aplicar estilos de una hoja de estilo en cascada (CSS). También puedes copiar y pegar código HTML de otra aplicación.
Vista Previa Muestra cómo aparecerá el contenido para los usuarios una vez publicado, lo que permite verificar que se vea como esperas antes de guardar los cambios.
03 Opciones del menú y su uso +

Las herramientas del Editor de Brightspace se agrupan en tres familias de acciones:

3.1 Formato de texto

Herramientas de formato de texto del editor.
Opción Uso Ícono
Formato (Paragraph Styling) Cambia el formato del texto a párrafo, encabezado (Heading), blockquote o código. Aplica una apariencia consistente y predefinida (fuentes, alineación, etc.) a cada párrafo.
Fuente Cambia la fuente del texto.
Tamaño de fuente Cambia el tamaño del texto.
Negrita Pone el texto en negrita. Atajo: dos asteriscos **ejemplo** generan negrita automáticamente al presionar Intro.
Cursiva Pone el texto en cursiva. Atajo: un asterisco *ejemplo* genera cursiva automáticamente al presionar Intro.
Formatos en línea Cambia el texto a subrayado, tachado, superíndice o subíndice.
Seleccionar color (color de fuente) Cambia el color del texto mediante el selector de color.
Alineación Cambia la alineación del texto.
Lista Permite elegir entre listas con viñetas o numeradas y ajustar la indentación (aumentar o disminuir).
Pintura de formato Resalta el texto para aplicar su formato o insertar un enlace.

3.2 Inserción de contenido

Herramientas para insertar contenido en el editor.
Opción Uso Ícono
Insertar objetos (Insert Stuff) Herramienta clave para agregar contenido: archivos, videos, enlaces, código incrustado, contenido desde Brightspace, desde tu dispositivo o desde herramientas de terceros (imágenes, videos, notas de video, documentos y mucho más). Los archivos cargados aquí se guardan en el área de Administrar archivos.
Insertar enlace rápido (Quicklink) Agrega enlaces a materiales del curso, actividades (como rúbricas) y contenido de terceros (URL externas).
Insertar imagen Agrega una imagen desde tu dispositivo, desde los archivos del curso o mediante una URL. El editor solicita automáticamente el texto alternativo (alt text) para garantizar la accesibilidad.
Otras opciones de inserción (botón +) Agrega elementos adicionales como ecuaciones, tablas, atributos, divisores, emoji, símbolos y códigos. También expande la barra de herramientas para ver opciones que no caben por el ancho de la pantalla.
Insertar tabla Permite añadir o modificar una tabla. Su uso asegura que las tablas se formateen de manera accesible.
Atributos Permite agregar o editar atributos como la orientación del texto, el título de la página y el idioma, desde Otras opciones de inserción > Atributos.

3.3 Herramientas de edición y accesibilidad

Herramientas de edición y accesibilidad del editor.
Opción Uso Ícono
Más acciones (puntos suspensivos) Expande la barra de herramientas y muestra opciones que podrían no verse de inmediato según el ancho de la ventana.

04 Editor de ecuaciones matemáticas y de química +

El Editor Gráfico de Ecuaciones (Graphical Equation Editor), accesible desde Otras opciones de inserción, permite añadir ecuaciones matemáticas dentro del Editor de Brightspace.

05 Formatos soportados +

El editor admite la entrada de cuatro tipos principales de ecuaciones:

  • Ecuación MathML: MathML es un estándar del W3C que utiliza XML para describir la notación matemática, capturando tanto su estructura como su contenido. Esto asegura su correcta visualización y el acceso por tecnologías de asistencia.
    Nota importante

    Brightspace almacena y muestra todas las ecuaciones en formato MathML, sin importar el formato con que las ingreses, y emplea el motor JavaScript MathJax para mostrarlas.

  • Ecuación LaTeX: LaTeX es un sistema de composición tipográfica basado en TeX que ofrece una sintaxis de texto para fórmulas matemáticas complejas. Brightspace guarda estas ecuaciones como MathML para mantener la consistencia y la accesibilidad.
    Nota

    LaTeX está disponible en la nueva experiencia de cuestionarios.

  • Ecuación gráfica (Graphical equation): Ofrece 10 pestañas de símbolos y diseños para construir ecuaciones matemáticas.
  • Ecuación química (Chemistry equation): Ofrece símbolos especializados para fórmulas de química inorgánica.

07 Creación y edición de ecuaciones +

7.1 Crear una ecuación (modo clásico)

  1. Accede al Editor de Brightspace en la herramienta donde deseas crear una ecuación.
  2. En la lista desplegable de Ecuación gráfica (el ícono Σ), elige la opción deseada: Ecuación gráfica, Ecuación química, Ecuación MathML o Ecuación LaTeX.
  3. En la ventana Insertar ecuación, escribe la fórmula con el teclado o seleccionando símbolos de las pestañas.
  4. Haz clic en Insertar.

7.2 Editar una ecuación existente

  1. Navega hasta el elemento del curso que contiene la ecuación y haz clic en Edit HTML (Editar HTML).
  2. Dentro del Editor de Brightspace, haz clic en el recuadro que contiene el ícono sigma.
  3. En la lista desplegable de Ecuación gráfica, selecciona el formato que deseas usar para editar (Gráfica, Química, MathML o LaTeX).
  4. Realiza los cambios necesarios.
  5. Haz clic en Insertar > Actualizar.
